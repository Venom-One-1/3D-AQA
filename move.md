# 动作要领-指标-反馈表

| 招式 | 关键帧 | 动作要领 | 可计算指标 | 可能反馈 |
| --- | --- | --- | --- | --- |
| 1.起势 | 预备式 | 两脚并拢，身体自然直立，头颈正直。两臂自然下垂，两手指尖轻贴大腿外侧。眼平视。 |  |  |
| 1.起势 | 分腿 | 身体自然直立，两脚开立，与肩同宽，脚尖向前；两臂自然下垂，两手放在大腿外侧；眼平看前方。 |  |  |
| 1.起势 | 双手上举 | 两臂慢慢向前平举，两手高与肩平，与肩同宽，手心向下。 | 肩臂夹角、肘关节角、肩部高度 | 手臂抬得偏低 / 肩部略有上提 |
| 1.起势 | 屈膝下蹲 | 上体保持正直，两腿屈膝下蹲；同时两掌轻轻下按至腹前，两肘下垂与两膝相对；眼平看前方。 |  |  |
